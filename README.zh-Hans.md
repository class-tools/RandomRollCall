# RandomRollCall

一个使用 Python 制作的小型 GUI 随机点名程序。

[简体中文](./README.zh-Hans.md) | [English](./README.md)

## 安装 / 使用

从 [Releases](https://github.com/ren-yc/RandomRollCall/releases) 下载 `.exe` 二进制文件。

双击运行 `.exe` 文件即可使用 “RandomRollCall”。
